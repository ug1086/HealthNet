Create Filters

Select the types of filters you want. "Date" will allow you to select a start date and an end date and display all actions occurring between those dates. "Username" will allow you to enter a username and will display all of the actions done on or by that username. "User Type" will allow you to select a user type (patient, nurse, doctor, admin) and will display all of the actions done by that user type. "Action" will allow you to select a specific action (logging in, logging out, creating an appointment, etc.) and will display all of the actions of that type. Checking multiple filters will allow you to apply multiple filters.
